# 📁 PASTA DE ENTRADA

## 🚀 Como usar:

1. **Coloque aqui** os arquivos Excel de empréstimos e pendências
2. **Execute** o ProcessadorBiblioteca.exe
3. **Aguarde** o processamento automático
4. **Verifique** a pasta "Saida" para os resultados

## 📊 Tipos de arquivo aceitos:

### ✅ Empréstimos:
- Arquivos com "emprestimo", "empréstimo", "loan" no nome
- Formatos: .xlsx, .xls

### ✅ Pendências:
- Arquivos com "pendencia", "pendência", "pending" no nome  
- Formatos: .xlsx, .xls

## 📋 O que acontece:

- ✅ **Arquivos processados** → Movidos para "Processados"
- ❌ **Arquivos com erro** → Movidos para "Erros"
- 📊 **Resultados** → Salvos na pasta "Saida"

## 🔍 Identificação automática:

O sistema identifica automaticamente o tipo de arquivo baseado no nome:
- "emprestimo" → Processado como empréstimos
- "pendencia" → Processado como pendências

---
💡 Dica: Use nomes descritivos nos arquivos para facilitar a identificação!
